# Fix Retargeter

Correção do plugin Mixamo Animation Retargeter para Godot 4.6.

[![PLUGIN](https://img.shields.io/badge/plugin-retargeter-lightgrey)](https://github.com/RaidTheory/Godot-Mixamo-Animation-Retargeter)  
[![GODOT](https://img.shields.io/badge/engine-godot-blue)](https://godotengine.org)

## Setup

Você pode:

1. Copiar o conteúdo do arquivo "mixamo_animation_retargeter.gd" direto do Github e colar dentro do arquivo com o mesmo nome, que fica na pasta do plugin, dentro de "addons" no Godot;
   OR
2. Baixar o arquivo "mixamo_animation_retargeter.gd" do Github e substituir o arquivo com o mesmo nome, que fica na pasta do plugin, dentro de "addons" no Godot.

## Teste

Uma vez realizado o passo "Setup", o plugin deve funcionar normalmente.
